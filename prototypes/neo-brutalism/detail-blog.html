CSS FRONTEND 2025

Why I stuck with
Vanilla CSS in 2025

PUBLISHED: SEPTEMBER 12, 2024
READ TIME: 6 MIN READ

TL;DR: Build tools come and go, but the browser platform remains. Here is why I removed Tailwind from my personal projects in favor of modern CSS variables and nesting.

In the ever-shifting landscape of frontend development, we often find ourselves oscillating between "separation of concerns" and "colocation." For the past five years, utility-first frameworks dominated my workflow. They were fast, efficient, and solved specificity wars.

But in 2025, the browser has caught up.

The Native Stack

With the introduction of native CSS Nesting and the `@scope` rule, many of the problems utility classes solved are now handled natively by the engine. We no longer need a build step to write maintainable styles.

"The best build tool is no build tool at all. The closer we get to the metal (or the browser engine), the more longevity our code has."

Consider this simple component. In a utility framework, it might look like a soup of classes. In modern vanilla CSS, we can structure it logically: 

/* Native CSS Nesting */
.card {
  background: var(--surface-1);
  border: 2px solid black;
  padding: 2rem;

  & h3 {
    font-weight: 800;
    margin-bottom: 1rem;
  }

  & button {
    background: black;
    color: white;
    
    &:hover {
        transform: translateY(-2px);
    }
  }
}

Performance Wins

By dropping the runtime overhead of CSS-in-JS libraries and the build-time complexity of huge utility generators, we ship smaller bundles. My personal site's lighthouse score jumped from 92 to 100 just by removing the styling framework.

Key Takeaways

  • CSS Variables are more powerful than pre-processor variables because they can react to the DOM.
  • Native Nesting removes the need for Sass/Less.
  • @scope solves the specificity wars without BEM.
  • Maintainability is about clarity, not just brevity.

I'm not saying frameworks are dead. For large teams, the constraints they provide are invaluable. But for a personal digital garden? Raw CSS is incredibly liberating.
